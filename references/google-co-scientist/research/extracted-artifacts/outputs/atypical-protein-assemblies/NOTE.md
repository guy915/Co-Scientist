# Atypical protein assemblies — generated hypotheses + reports

This is the Co-Scientist run on the **AlphaFold3-scored structural-novelty
(SNI)** goal for unconventional NRC-NLR resistosomes. It is the richest single
example of the system's generated outputs, so it is included here in full:

- `research-goal.md` — the research goal fed to the system (the run's input).
- `hypotheses/` — **19 generated hypotheses** (Generation-agent SNI proposals;
  the longer ones carry their full review appendices).
- `reports/` — the **full reports**: `research-overview.md` (the Meta-review
  synthesis) and `top-ranking-hypotheses*.md` (the tournament comparison).

These files are an **exact copy** of the already-organized case study under
`../../../supplements/ai-guided-discovery-of-atypical-protein-assemblies/`;
that original folder remains the source of truth.
